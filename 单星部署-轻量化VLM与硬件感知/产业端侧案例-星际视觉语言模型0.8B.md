# 产业端侧案例：Stellaris-VL-0.8B

> **在 `单星部署-核心文献.md` 中的建议引用位置**：紧接“## 7. Chu et al.：MobileVLM”之后、进入 AWQ 之前。它不纳入 16 篇精读编号，也不作为星载部署或方法有效性的核心证据；其作用是补充当前产业端侧 VLM 向 0.8B 级模型、开放词汇感知与本地离线运行收敛的趋势。

- **来源与性质**：极视角于 2026 年 6 月 22 日发布的厂商文章《小参数，大能力｜星际视觉语言大模型再进化，0.8B轻量版正式发布》。该资料属于产品发布材料，不是同行评议论文、模型卡或可复现实验报告。
- **模型与能力主张**：Stellaris-VL-0.8B 面向端侧与嵌入式设备，聚焦开放词汇目标检测（OVD）、指代表达理解（REC）和视觉问答（VQA）。文章称模型采用轻量 LLM 底座与 ViT 视觉编码器，并结合知识蒸馏、SFT 监督微调与 RL 后训练。
- **端侧部署主张**：文章称模型可在终端本地离线运行，面向无人机、移动巡检机器人、车载/船载/矿载视觉等设备；其宣称目标包括秒级图像解析、长期连续识别吞吐稳定和降低本地部署门槛。
- **评测信息**：文章展示 ODinW13 开放词汇检测、指代表达定位及“1000+真实产业场景”的产业综合检测集结果，但未公开完整测试协议、基线配置、具体数值表、模型权重、代码或可复现实验细节。
- **星上关联**：OVD、REC 与 VQA 可分别映射到星上开放目标筛选、区域定位和语义解释；0.8B 级模型也可作为单星轻量 VLM 模型规模下界的产业参考。然而，该文章没有公开目标芯片、RAM、视觉 token、上下文长度、量化位宽、端到端时延、峰值内存、能耗、温度、辐射可靠性或遥感/SAR/多光谱实验，不能据此推导其适合卫星部署。
- **建议写入 Ch.5 的一句话**：除 MobileVLM 等论文路线外，产业界也开始推出 0.8B 级端侧 VLM，将开放词汇检测、指代表达理解和视觉问答置于同一轻量模型中；这说明小参数多模态模型正向本地部署收敛，但其具体的硬件预算、量化路径和长期可靠性仍需公开工程数据验证。
- **证据等级**：产业趋势线索；不进入 A（在轨）、B（飞行代表环境）、C（端侧可复现实测）、D（论文算法）证据分级。

## 适合与 MobileVLM 对照的维度

| 维度 | MobileVLM | Stellaris-VL-0.8B |
| --- | --- | --- |
| 模型规模 | 1.4B / 2.7B | 0.8B（厂商披露） |
| 核心任务 | 通用 VQA、描述、视觉推理与对话 | OVD、REC、VQA |
| 可公开机制 | LDP 视觉 token 下采样、训练与端侧实测 | 轻量 LLM + ViT、蒸馏/SFT/RL 的概述 |
| 运行证据 | Snapdragon 888 与 Jetson Orin 实测 | 宣称端侧/嵌入式可运行，硬件与指标未公开 |
| 在本综述中的定位 | 核心学术基线 | 产业趋势补充 |
