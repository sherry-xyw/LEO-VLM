# 产业与任务案例：在轨持续适应

本目录仅收录可用于说明工程需求、平台能力、任务更新流程或在轨运行状态的任务方与产业资料。它们不替代原始算法论文，也不应单独支撑“某算法已可在轨应用”的结论。

## 计划收集的材料类型

- **在轨模型更新案例**：模型、任务头、规则库、配置或知识库在任务期内发生版本变化的官方记录；
- **星座协同案例**：星间/星地更新、任务重配置、轨道内数据汇聚的任务方资料；
- **载荷与平台资料**：可用训练内存、储存、功率、热控、链路窗口和安全启动/回滚能力；
- **产业端 RAG / Edge AI 案例**：仅作为“非参数知识更新”和“本地模型更新”的工程趋势线索。

| --- | --- | --- | --- | --- |

## 使用边界

- 产品规格、新闻稿和厂商案例只作为 E 级趋势线索；
- 任务方技术报告、在轨演示论文或官方工程文档可作为 B/A 级工程证据，但需注明任务日期、实际载荷与指标口径；
- 不以“支持 OTA / 在轨重配置”推导“支持在轨 VLM 微调”。
